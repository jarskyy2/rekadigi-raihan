# Dokumentasi

# Tech Stack Nextjs, Express, and PostgreSQL

## Installation

### 1. Clone Repo

```bash
git clone <repository_url>
cd <repository_folder>
```

### 2. Backend Setup

```bash
cd api
npm install
npm run migrate:up
npm run seed
npm run dev
```

Cek pada `http://localhost:8000`

### 2. Frontend Setup

```bash
cd ../web
npm install
npm run dev
```

Cek pada `http://localhost:3000`
